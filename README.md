# OfficeRoad

*Your agents come alive.*

Animated 2D office where your 27 convoy agents walk, work, and collaborate — visible, alive, and always on the clock.

## The Ride

Step into OfficeRoad. Open the office, and there they are. Cecilia at her desk, Eve researching in the corner, Pixel sketching on the whiteboard, Cadence arguing about the playlist. Your 27 agents aren't just running in the background — you can see them. Watch them collaborate. Drop a task on one of their desks. They get to work.

## What It Does

A real-time animated 2D office environment rendered in the browser. Each agent has a desk, personality animations, and a live status indicator showing what they're currently working on. OfficeRoad is the visual layer on top of RoadTrip — same agents, but now you can see them.

## Integrations

| Service | Role |
|---------|------|
| **Cloudflare Workers** | Office runtime — real-time agent state at the edge |
| **Cloudflare D1** | Agent positions, desk assignments, active task state |
| **RoadTrip** | Live feed of agent activity — OfficeRoad visualizes the convoy |
| **CarPool** | Routes tasks dropped on agent desks to the right AI provider |
| **RoadChain** | Every completed task stamped and logged to the ledger |
| **RoadCoin** | Agents earn ROAD for tasks completed at their desks |
| **Ollama** | Local inference for on-device agent activity (Pi fleet) |
| **Workers AI** | Cloud fallback for agent responses |

## Features

- Real-time animated sprites for all 27 agents with personality-driven idle animations
- Live status badges: thinking 💭, working ⚙️, waiting 💤, in-meeting 🗣️
- Drag tasks onto any agent's desk to assign work
- Agent speech bubbles showing live AI responses
- Meeting room — gather agents for a group task with one click
- Whiteboard that syncs with BlackBoard canvas
- Notifications when an agent completes a task or needs input
- Night mode — watch agents go home at midnight and return at 8am
- Click any agent to open a direct RoadTrip chat
